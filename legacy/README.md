# Legacy Upstream Workflow

This directory contains the original R Markdown workflow and supporting files
from the upstream `Ttantivi/NBA_Predict` repository.

These files are preserved for traceability:

```text
NBA_data_preprocessing.Rmd
NBA_Predict_Models.Rmd
NBA_Predict_Other_Models.Rmd
Method_Details.pdf
Images/
```

The active reproducible project pipeline is now implemented in Python under
`src/nba_predict` and is run through the `nba-predict` command line interface.

Use this directory only as a reference for the original implementation and model
description.
